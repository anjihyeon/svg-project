# svg-project

svg를 이용해 barchart로 시각화.<br>
.csv파일을 input해서 그에 맞는 barchart가 나오도록 만들었습니다.

## 기능
- .csv파일 innput
- svg 너비,높이 사이즈 변경
- bar 색깔 변경
